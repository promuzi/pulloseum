## 1. 스탯 시스템

### 1-1. 전투 스탯 8종

| 스탯 | 키 | 설명 |
|---|---|---|
| 생명력 | `hp` | 최대 체력 |
| 공격 | `atk` | 일반·물리 위력 계수의 기반 |
| 방어 | `def` | 받는 피해 고정 차감 |
| 기동성 | `spd` | 선공 판정 + 회피율 |
| 적중 | `acc` | 명중률(회피 상쇄) |
| 속성 | `elem` | **속성기**에만 가산되는 추가 피해치 |
| 치명 | `critRate` | 치명타 확률(%) · 계수는 `critMult` 고정 ×1.5 |
| 에너지 | `energy` | 스킬 코스트 자원(최대치) |

### 1-2. 베이스 스탯 = 속성 + 타입 (합산)

종 1마리의 기초 스탯은 **`ELEMENT_STATS[속성] + TYPE_STATS[타입]`**. 두 표만 고치면 35종 전체가 일괄 조정됩니다.

### 1-4. 베이스 → 실제 전투 스탯 변환 (`calcStats`)

```
hp   = (base.hp  + 잉여유전자×6) × 개체배율 × 생장배율 × (1 + 강화hp ×0.05)
atk  = (base.atk + 유전자atk×2) × 개체배율 × 생장배율 × (1 + 강화atk×0.05)
def  = (base.def + 유전자def×2) × 개체배율 × 생장배율 × (1 + 강화def×0.05)
spd  = (base.spd + 유전자spd×1.5)× 개체배율 × 생장배율 × (1 + 강화spd×0.05)
acc      = clamp(86 + 강화acc×2 + (개체배율−1)×8, 70, 99)
elem     = (14 + 강화elem×5) × 생장배율
critRate = clamp(5 + 강화crit×2, 0, 75) %
energy   = clamp(기본3 + 강화energy, 1, 9)
```
- **개체배율(`mult`)** — 종자 생성 시 스탯별 0.75~1.45 무작위. 평균으로 개체등급 S/A/B/C 결정. 12% 확률 각성(한 스탯 +0.3).
- **생장배율(`GROWTH_STAT_MULT`)** — 씨앗0.9 / 새싹1.0 / 유체1.16 / 성장체1.24 / 성체1.32 / **완숙체1.5**.
- **크레딧 강화(`up`)** — 레벨당 +5%(스탯), elem은 +5씩, crit는 +2%씩, energy/acc는 정수 가산. 곱연산이라 성장해도 가치 유지.

## 4. 데미지 공식 (`applySkill`)

```
1. 명중 판정:   회피% = clamp(30 + (방어자 기동성 − 유효적중), 5, 90)
                유효적중 = 적중 + (광역 +10 / 단일 −7) + accBonus
2. 기본 피해:   raw = 공격×위력%/100  +  [속성기면] 속성수치 × 상성배율
3. 방어 차감:   afterDef = raw − 방어×(1−관통)
                dm = max(1, max(raw×0.25, afterDef))    ← 원 피해의 25% 하한선
4. 치명타:      확률 = critRate% + skill.critBonus + passive.critBonus → 적중 시 ×1.5
5. 보정:        ×다음공격버프 → ×방어태세 → ×피해감소 → ×속성저항 → ×변동(0.97~1.03)
6. 보호막 흡수 → 체력 차감 → (치명상 생존 패시브) → 흡혈/강탈/독/반사/반격
```

- **기본 공격 위력 135** (저위력기가 방어로 1뎀 되는 문제 보정용으로 상향됨).
- **25% 하한선** 덕분에 방어가 높아도 최소 원 피해의 ¼은 들어감.
- 속성기는 `위력 + 속성수치(약14~21)×상성`이라 상성 맞으면 폭발적.

## 8. ⚠️ 밸런스 검토 포인트 (현재 발견된 것)

> ⚠️ **2026-07-02 전투 코어 3v3 재설계(#17) 이후 전제 변경:** ① **에너지 = 팀 공유 풀, 시작 2·매 턴 +1·최대 8**(시작 만땅 폐지 — 고코스트 스킬 가치·`energyGain` 가치 상승). ② **로드아웃 8장=덱, 손패 4장 사이클**(한 턴에 손에 있는 4장+숨 고르기+교체만 선택 가능). ③ 3v3 리그는 교체 심리전 포함 — 스킬 밸런스 평가 시 "손패에 잡힐 확률(4/덱크기)"과 공유 에너지를 함께 볼 것.

수정 지시 시 참고할 만한 잠재 이슈:

1. ~~**잠재특성 20종 패시브가 전투에서 작동 안 함.**~~ **(결론: 의도적 미반영 — 이슈 아님, 2026-06-26)** `plantPassives()`가 전부 0을 반환해 재생·흡혈·반사·방깎 등 특성 패시브가 전투에 미반영인 것은 **의도된 설계**다. 잠재특성 20종은 "성장 단계 스킬 해금 분류"로만 쓰고 패시브는 **예약(미구현)** 상태로 둔다(로드맵 §3). 실제 전투 보정은 변이 카드 효과로만 적용. → 향후 특성 패시브를 살릴지는 별도 방향 결정 대상이며, 현 시점 버그/이슈 아님.

2. ~~**종간 BST 격차 ~42.**~~ **(✅ 완화 2026-07-02)** 불 `ELEMENT_STATS` hp40→46·atk14→15(+7), 바람 hp40→44·def5→7(+6) 상향 → 최저-최고 격차 ~42→~35. 컨셉(불=화력·바람=최속 유리몸)은 유지, 과벌점만 완화.

3. ~~**무기 스킬 고위력.**~~ **(✅ 견제 2026-07-02)** `gradeCardSkill`에서 무기(noEnergy) 스킬의 등급 위력 스케일을 60%로 완충: `1+(mult−1)×0.6`. S 내려찍기 215×1.9=409 → 215×1.54=**331**. 저등급(D 0.8)은 오히려 소폭 상향(0.88)이라 하위 무기 체감 개선. 기본 위력·횟수 체계는 불변.
