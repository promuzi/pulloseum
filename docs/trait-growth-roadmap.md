## 0. 현재 구현 요약 (출발점)

코드(`index.html`)에 이미 들어 있는 것:

- **종(SPECIES) 7종** — `flame/aqua/thorn/gaia/windy/spark/frost`. 각자 속성 1개와 기본 스탯(hp/atk/def/spd) 보유.
  - ⚠️ 이건 아래 "다음 명령"의 종자 종류(풀/덩쿨/꽃/선인장/나무)와는 **다른 축**이다(현재는 속성-종 위주).
- **형질(FORMS) 3종** — `normal(일반형)/pred(포식형)/weapon(무기형)`. 종자 획득 시 `rollForm()`으로 분화.
  - `PRED_TYPES`: 흡혈(vamp)/능력치 흡수(absorb)
  - `WEAPONS`: 바이오 블레이드(blade)/씨앗 캐논(cannon)/쏜 실드(shield)
- **특성 20종** — 전투 패시브(재생/흡혈/반사/방깎 등) + 시그니처 스킬. `TRAIT_SKILL_MAP`, `plantPassives()`.
- **스킬 로드아웃** — 식물이 속성+특성+생장단계로 스킬을 얻고 **최대 6개**를 장착해 전투.
- **생장 5단계** — 씨앗→새싹→유체→성체→완숙체. 단계 상승 시 새 스킬 해금 + 스탯 배수.

## 1. 목표 특성 시스템

### 1.1 공통 규칙
- **특성 장착칸은 총 5개.**
- 식물이 **보유한 특성에 해당하는 특성 카드만** 장착 가능.
- 특성은 **형질(form)별로 장착 가능한 카테고리가 다르다**(아래 표).

| 형질 | 장착 대상 카드 | 부여 방식 | 비고 |
|---|---|---|---|
| 일반형 | (무특성) 공통 특성만 | 패시브 | 무특성 식물 |
| 포식형 | DNA | "포식" 스킬 영구 획득 + 강화 | 포식 스킬 중심 |
| 무기형 | 무기 | 장착 시 추가 스킬 자동 장착 | 무기별 전용 스킬 |
| 독성형 | 물약 | 추가 스킬 자동 장착 + 독성(속성) 보정 | 독속성 계수 |
| 포자형 | 포자 | 매 턴 종료 시 발동 패시브(중복 가능) | 확률 발동 |
| 발광형 | 엽록체 | 에너지·체력 회복 보정/패시브 | 자원 회복 중심 |
| 용족 | 미정 | 미정 | 추후 설계 |

### 1.2 형질별 상세

#### 일반형 (= 무특성)
- 공통으로 장착 가능한 특성만 장착 가능.
- 공통 특성 예:
  - **튼튼한 세포벽**: 방어력 증가
  - **가시 줄기**: 상대 공격 시 일정 배율로 데미지 반사

> 공통 특성은 다른 형질도 함께 장착 가능한지(혼합) 여부는 추후 결정.

#### 포식형
- 특성 칸에 **DNA** 장착.
- 해당 특성으로 **"포식" 스킬을 영구 획득**, 특성 장착 시 포식 스킬이 강화됨.
- 특성 예:
  - **진액 흡수**: "포식" 스킬에 일정 흡혈 부여
  - **엽록체 흡수**: 에너지 강탈
  - **미토콘드리아 흡수**: 공격력 강탈

#### 무기형
- 특성 칸에 **무기** 장착.
- 장착 시 **추가 스킬이 자동 장착**됨.
- 특성 예:
  - **검**: 검 베기 스킬
  - **방패**: 방패 방어 스킬
  - **도끼**: 내려찍기 스킬
  - **총**: 총 쏘기 스킬

#### 독성형
- 특성 칸에 **물약** 장착.
- 장착 시 추가 스킬 자동 장착 + **독성(속성) 보정값 추가**.
- 특성 예:
  - **강산**: 강산 뿌리기 스킬
  - **독성가스**: 가스 유출 스킬
  - **독가시**: 공격 시 독속성 계수 부여(모든 공격 스킬에 추가 데미지)

#### 포자형
- 특성 칸에 **포자** 장착.
- **매 턴 종료 시 발동하는 패시브.** 여러 포자 중복 발생 가능.
- 특성 예:
  - **마비 포자**: 매 턴 일정 확률로 자동 사용 → 일정 확률로 상대 턴 마비
  - **중독 포자**: 매 턴 일정 확률로 자동 사용 → 일정 확률로 상대에게 데미지
  - **방해 포자**: 매 턴 일정 확률로 자동 사용 → 일정 확률로 다음 턴 내 방어력 증가

#### 발광형
- 특성 칸에 **엽록체** 장착.
- 특성에 따라 **에너지·체력 회복 스킬 보정, 패시브 회복** 등이 적용.
- 특성 예:
  - **엽록체 증가**: 턴당 에너지 회복량 증가(예: 1→2)
  - (추가 예정)

#### 용족
- 미정. 특성 예 미정.

## 2. 특성 획득 규칙 (2026-06-16 확정·구현)
- 특성 카드는 **기본 지급 없음.** 오직 **① 상점 뽑기(🎴 특성 카드 뽑기, `grantRandomTraitCard`)**
  와 **② 탐사 보상(`rollExplorationRewards` 드롭)** 으로만 획득한다.
- 획득한 카드는 계정 보유 인벤토리 `state.trait_cards`에 쌓이며, 식물은 **보유 중이고 형질에 맞는
  카드만** 5칸에 장착 가능(`ownedFormCards`).
- **포식형:** "포식" 스킬을 **카드 없이 기본 보유**(`skill_card_predation`). 기본 포식엔 흡혈·강탈 **없음**.
  DNA 카드(진액 흡수=흡혈 / 미토콘드리아=공격력 강탈 / 엽록체 흡수=에너지 강탈)로 **포식만** 강화된다.
- (향후) 변이율이 높은 개체의 추가 특성 획득 로직은 미구현.

## 3. 다음 명령 로드맵 (추후 설계·구현 예정)

아래는 사용자가 "추후 업그레이드 방향"으로 지정한 항목. 지금은 준비만, 구현은 다음 단계.

1. **종자 종류별 특징·스탯** — 풀 / 덩쿨 / 꽃 / 선인장 / 나무.
   - 각 종류의 컨셉, 기본 스탯 성향(공/방/속/체) 정의.
   - ⚠️ 현재 코드의 SPECIES(속성-종 7종)와 이 "종자 종류" 축을 어떻게 통합/분리할지 결정 필요.
2. **생장 단계별 스킬·스탯 변화 구체화** — 씨앗→새싹→유체→성체→완숙체 각 단계의
   해금 스킬, 스탯 배수, 변화 곡선을 구체적으로 정의.
3. **이름·디자인 부여 및 일치** — 종자 / 속성 / 특성 / 생장 단계 각각에 일관된 이름과
   비주얼(SVG/아이콘/색)을 부여하고 서로 맞물리게 정리.
