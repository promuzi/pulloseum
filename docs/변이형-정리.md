# 변이형(변이 종류) — 정정 메모

> 작성: 2026-06-27 · 작업 중 잘못 이해했던 "변이 종류"를 바로잡아 박제. 다음에 또 틀리지 않도록.

## 변이형은 5개 (비버섯 종 기준)

| 변이형 | 코드 | 아이콘 |
|---|---|---|
| 일반 | `normal` | 🌱 |
| 포식 | `pred` | 🦷 |
| 무기 | `weapon` | 🗡️ |
| 독성 | `toxic` | ☠️ |
| 용족 | `dragon` | 🐉 |

- **포자(spore) 🍄 = 버섯 전용.** 비버섯 종은 포자 변이를 갖지 않는다. (버섯은 포자, 일부는 포자+다른 변이의 이중 변이.)
- **발광(lumen) 💡 = 폐지.** 신규 생성 없음(`rollForm`에서 제거). 정의·기존 legacy 식물만 잔존.
- **base = 비변이.** 변이형이 아니다. 태생 변이(`baseVariants`)나 변이 카드가 없는 보통 종.

## 종 구성 공식

```
종 = 식물 타입 × 속성 × (변이형 5 또는 비변이)
```
- **식물 타입(5):** 목본 / 화초 / 다육 / 덩굴 / 버섯
- **속성(7):** 불 · 물 · 풀 · 대지 · 바람 · 번개 · 빙결
- **변이형(5):** 일반 · 포식 · 무기 · 독성 · 용족  (+ 비변이 base / 버섯=포자)

예) 덩굴 타입 = **base(비변이) + 5 변이형 = 6 그룹 × 7속성 = 42종**.

## 내가 잘못 이해했던 것 (정정)

1. ❌ "**6 변이형**"이라 셈 — base를 변이형에 포함시킴.
   → ✅ 변이형은 **5개**, **base는 비변이**(변이형 아님).
2. ❌ 같은 변이형 안에서 속성 차이를 **색만**으로 처리.
   → ✅ 속성은 색뿐 아니라 **외형 해석**도 달라야 한다(불=이글거림, 빙결=결정 등).
3. ❌ 화분을 식물과 같은 레이어로 오해.
   → ✅ 화분은 `spriteStack`에서 **별도 레이어**(`.ss-pot` / `.ss-plant`)로 렌더되며 픽셀 단위도 따로 잡힌다.

## 근거 (코드)
- `FORMS` 정의(7개: normal/pred/weapon/toxic/spore/lumen/dragon) — `index.html`.
- 발광 폐지: `rollForm`(`lumen ... 제거 — 발광형 폐지`), 발광 보급상자 제거 주석.
- 포자=버섯 전용: 포자 종은 버섯 타입에만, `mushroom.spore_burst` 단계 스킬.
- 상위 안내(CLAUDE.md): "변이형 6종(무기/포식/독성/포자/용족/일반). 발광(lumen) 폐지. 포자형=버섯 전용." → 즉 **비버섯이 가질 수 있는 변이형은 5개**.
