# 풀로세움 — 전투 · 생장 · 스탯 공략집 (밸런스 시트)

> **목적:** 유저는 "어떻게 키우고 싸우나"를, 개발자는 "어느 수치를 만지면 되나"를 한 장에서 보게 정리한 문서.
> 모든 수치는 `index.html` 안의 데이터/공식에서 뽑았으며, **수정 위치(파일 내 상수명)** 를 함께 적었다.
> 피드백은 각 섹션 끝의 〈밸런스 메모〉 줄에 적거나, 표의 수치를 직접 고쳐 의견을 달면 된다.
>
> 기준일: 2026-06-18 · 출처: `index.html`

## 1. 식물(종) 기본 스탯

`SPECIES` (index.html). 종마다 **고정 속성**과 기본 스탯을 가진다. (외형·이름·속성이 종에 묶여 있음)

| 종 | 속성 | 컨셉 | HP | 공격 | 방어 | 기동성 |
|---|---|---|---:|---:|---:|---:|
| 플레임모스 | 🔥불 | 공격 특화 | 90 | 26 | 14 | 18 |
| 아쿠아벌브 | 💧물 | 방어 특화 | 120 | 18 | 22 | 14 |
| 쏜위드 | 🌿풀 | 속도 특화 | 100 | 22 | 16 | 24 |
| 가이아록 | 🪨대지 | 중장갑 | 115 | 20 | 24 | 10 |
| 윈드리프 | 🌪️바람 | 초고속 | 85 | 21 | 13 | 30 |
| 스파크번 | ⚡번개 | 화력(유리대포) | 80 | 28 | 12 | 26 |
| 프로스트베리 | ❄️빙결 | 지구력 | 110 | 24 | 18 | 11 |

〈밸런스 메모〉 스파크번이 HP 80 + 방어 12로 가장 물러 "유리대포" 역할이 분명. 가이아록은 기동성 10이라 거의 항상 후공.

## 2. 속성 상성

`ELEMENTS` / `eff(a,d)`. 강점=**1.5배**, 약점=**0.67배**, 그 외 1배.

| 공격 속성 | 강하게 때림(1.5×) | 약점(0.67× 받음) |
|---|---|---|
| 🔥불 | 풀 · 빙결 · 바람 | 물 · 대지 |
| 💧물 | 불 · 대지 | 풀 · 빙결 · 번개 |
| 🌿풀 | 물 · 대지 | 불 · 바람 · 번개 |
| 🪨대지 | 불 · 번개 · 빙결 · 바람 | 풀 · 물 |
| 🌪️바람 | 풀 | 대지 · 불 |
| ⚡번개 | 물 · 풀 | 대지 · 빙결 |
| ❄️빙결 | 물 · 번개 | 불 · 대지 |

〈밸런스 메모〉 대지가 4종(불·번개·빙결·바람)을 1.5배로 때려 공격 커버리지가 가장 넓다. 바람은 강점이 풀 1종뿐이라 상성상 약함.

## 3. 생장 단계

`GROWTH_STAGE_ORDER` / `GROWTH_STAGE_NAMES` / `GROWTH_STAT_MULT` / `GROWTH_EXP_REQUIREMENTS`.

단계가 오르면 **모든 전투 스탯에 배율(g)** 이 곱해지고, **새 스킬이 해금**된다. (초기 식물은 새싹부터 시작)

| 단계 | 스탯 배율 | 다음 단계까지 필요 경험치 | 해금 스킬(누적) |
|---|---:|---:|---|
| 씨앗 seed | ×0.90 | 18 | 기본 공격 |
| 새싹 sprout | ×1.00 | 45 | + 방어 태세, **속성기** |
| 유체 juvenile | ×1.16 | 90 | + 광합성, 기력 집중, 속성 성장스킬① |
| 성장체 growing | ×1.24 | 125 | + 생장 가속, 속성 성장스킬② |
| 성체 mature | ×1.32 | 160 | + 속성 성장스킬③ |
| 완숙체 evolved | ×1.50 | (최종) | + 속성 성장스킬④ |

- **경험치 획득:** 전투 승리(`gainGrowthExp`) / 물·비료·영양제 소모품 / 물주기·비료주기 액션.
  - 소모품 경험치: 물 8 · 비료 20 · 영양제 45 (상점 `SHOP_GROWTH_CONSUMABLES`)
- 단계 상승 시 **진화 연출 팝업**(스탯 증감 비교 + 새 스킬 표시). 단, 전투 승리로 오른 경우엔 결과창과 겹치지 않게 팝업 생략.

〈밸런스 메모〉 완숙체 ×1.5는 새싹 대비 +50% 스탯. 속성별 성장스킬 4종은 `ELEMENT_GROWTH_SKILLS`에서 약→강 순으로 단계마다 1개씩 해금.

## 4. 최종 스탯 계산식

`calcStats(p)`. 종 기본값에 유전자 분배·개체 배율·생장 배율·강화 레벨을 곱/합산.

```
HP   = round((기본HP  + 잉여유전자×6) × 배율_hp  × g) + 강화_hp ×15
공격 = round((기본공격 + 분배_atk×2)  × 배율_atk × g) + 강화_atk×4
방어 = round((기본방어 + 분배_def×2)  × 배율_def × g) + 강화_def×4
기동 = round((기본기동 + 분배_spd×1.5)× 배율_spd × g) + 강화_spd×3
적중 = clamp(86 + 강화_acc×2 + round((배율_atk−1)×8), 70, 99)
속성 = round((14 + 강화_elem×5) × g)      // 원소 스킬 피해에 가산
치명률 = clamp(5 + 강화_crit×2, 0, 75) %   치명계수 = ×1.5 고정
에너지 = clamp(기본3 + 강화_energy, 1, 9)
```

- **g** = 생장 배율(§3). **유전자 예산** `GENE_BUDGET = 15`(공/방/기동에 분배, 잉여는 HP로).
- **개체 배율** `rollMult()` = 0.75~1.45 무작위, 12% 확률로 한 스탯 +0.3 각성. 평균으로 등급(C/B/A/S) 결정.

〈밸런스 메모〉 강화 1레벨당 HP+15 / 공·방+4 / 기동+3. 잉여 유전자는 HP로만 가니 극단 분배 시 물몸 주의.

## 5. 전투 데미지 공식

`applySkill`. 한 턴에 양쪽이 1회씩 행동(기동성으로 선·후공 결정).

```
일반피해 = 공격 × 위력% / 100
속성피해 = (s.elem일 때) 속성수치 × 상성배율(1.5 / 1 / 0.67)
원피해   = 일반피해 + 속성피해
실피해   = max(1, max(원피해×0.25, 원피해 − 방어×(1−관통)))   ← 방어로 깎여도 원피해의 25%는 보장
치명타   = (치명률 + 스킬보너스 + 패시브) 확률로 ×1.5
+ 방어태세 배율 · 피해감소 · 속성저항 · 보호막 흡수 적용
최종     = round(실피해 × 0.97~1.03 난수)
```

- **명중/회피:** 회피율 = `clamp(30 + 방어자기동 − (공격자적중 + 광역/단일보정), 5, 90)`. 광역 +10 적중, 단일 −7 적중.
- **선공 확률:** `clamp(50 + (내기동 − 상대기동), 5, 95)%`.
- **에너지:** 매 턴 양쪽 +1. 스킬은 비용만큼 차감. 무기 스킬은 **에너지 무소모 + 경기당 횟수 제한**.

〈밸런스 메모〉 **25% 하한선**이 핵심 보정. 기본 공격 위력 100→135로 올린 뒤 이 하한선 덕에 방어형 상대에게도 기본기가 통한다. 원소·잠재기는 위력이 높아 하한선이 거의 안 걸림.

## 6. 스킬 일람

스키마: `{ power 위력, cost ⚡, kind, elem 속성기여부, pierce 방관, aoe 광역, single 단일, dot 지속피해, heal 회복, selfBuff/enemyDebuff, lifesteal 흡혈 }`
> **위력 N** = 공격력의 N%라는 계수(실제 피해 아님).

### 6-1. 공용 스킬 `UNIVERSAL_SKILLS`
| 스킬 | ⚡ | 효과 |
|---|:--:|---|
| 👊 기본 공격 | 0 | 위력 135 · 단일 |
| 🛡️ 방어 태세 | 0 | 받는 피해 50%↓ · ⚡+1 |
| ☀️ 광합성 | 2 | 체력 28% 회복 |
| 🌀 기력 집중 | 0 | ⚡2 충전 |
| 📈 생장 가속 | 2 | 기동 25%↑(3턴) |

### 6-2. 속성기 `ELEMENT_SKILLS` (속성당 1종)
- 위력 **170 · 광역 · ⚡2 · 속성 상성 + 속성수치 가산**. (화염분사/고압물대포/가시폭풍/암석붕괴/진공칼날/천둥직격/절대영도)

### 6-3. 잠재특성 스킬 `TRAIT_SKILLS` (생장단계 해금분, 발췌)
| 스킬 | ⚡ | 효과 요약 |
|---|:--:|---|
| 🌟 광합성 과충전 | 2 | 체력22%↑ + 공격15%↑(3턴) |
| 🟢 공명 방벽 | 2 | 피해55%↓ + 방어20%↑ |
| 🧫 산성 분사 | 2 | 위력115 광역 + 방어20%↓ + 중독 |
| 🪨 철피 경화 | 2 | 방어35%↑ + 체력10%↑ |
| 🌵 가시 사출 | 2 | 위력110 + 출혈(3턴) |
| 🪐 중력 붕괴 | 3 | 위력130 광역 + 적 기동30%↓ |
| ⚡ 플라즈마 개화 | 3 | 위력165 광역 + 치명15% |
| 🦷 포식 아귀 | 3 | 위력130 + 50% 흡혈 |
| 🌞 태양 폭발 | 3 | 위력150 광역 + 공격20%↑ |
| 🔮 정신 간섭 | 3 | 위력110 + 적 공격18%↓ |
| ⚙️ 금속 관다발 | 2 | 위력120 + 방어 50% 무시 |

> 전체 20종은 `TRAIT_SKILLS` 참조. 속성별 해금 순서는 `ELEMENT_GROWTH_SKILLS`.

### 6-4. 로드아웃
- **최대 6칸**(`MAX_LOADOUT = 6`). 기본 공격은 항상 포함.

## 7. 변이형(폼) & 변이 카드

식물은 7종 변이형 중 하나로 분화(`rollForm`). 변이형마다 장착 가능한 카드 카테고리가 다르다.

| 변이형 | 분화 확률 | 카드 종류 | 특징 |
|---|:--:|---|---|
| 🌱 일반 | ~22% | 공통 | 무난 |
| 🦷 포식 | 16% | DNA | 포식 기본 스킬(위력150) + 흡혈·강탈 강화 |
| 🗡️ 무기 | 16% | 무기 | **에너지 무소모 + 경기당 횟수 제한** 강타 |
| ☠️ 독성 | 14% | 물약 | 공격에 독성 추가 피해 |
| 🍄 포자 | 14% | 포자 | 매 턴 종료 시 마비/중독/방해 발동 |
| 💡 발광 | 14% | 엽록체 | 턴당 에너지·체력 회복 |
| 🐉 용족 | 4% | 용 DNA | 희귀(콘텐츠 미정) |

**공통 카드는 용족 제외 전부 장착 가능.** 카드 장착칸 최대 5개.

### 카드 등급 S~D `CARD_GRADE_META`
등급↑ → 주효과 계수↑ + 서브 특성 수↑. 보급상자 개봉 시 등급 무작위(`rollCardGrade`).

| 등급 | 주효과 계수 | 서브 특성 수 | 출현 가중치 |
|:--:|:--:|:--:|:--:|
| S | ×1.90 | 4 | 5 |
| A | ×1.55 | 3 | 11 |
| B | ×1.30 | 2 | 20 |
| C | ×1.00 | 1 | 30 |
| D | ×0.80 | 0 | 34 |

- **서브 특성 풀** `SUB_FX`: 공격+10% / 방어+12% / 기동+14% / 체력+10% / 치명+10% / 흡혈+10% / 반사+12% / 재생+4% / 에너지+1 / 공격 시 적 방어 12%↓.
- 무기 카드 스킬은 등급에 따라 위력·사용횟수(uses)도 스케일(`gradeCardSkill`).

〈밸런스 메모〉 무기형은 ⚡소모가 없어 폭발력이 높지만 횟수 소진 후엔 기본기에 의존. 포자형은 운(발동 확률) 의존도가 큼.

## 8. 상태이상 / 지속효과

`tickStatuses` — 매 턴 종료 시 처리.
- **지속피해(DOT):** 🟣중독 · 🩸출혈 · 🔥화상. 턴당 `공격 × pct` 고정 피해, turns 동안.
- **버프/디버프:** 스탯 ±%, turns 경과 시 소멸. `effStat`로 실시간 반영.
- **재생:** 패시브/카드의 `regenPct`로 매 턴 HP 회복.
- **포자(매 턴):** 마비(다음 턴 행동 불가) / 중독 / 방해(자버프).
- **반사·반격:** 피격 시 `reflectPct` 비례 반사, 방어 태세 카운터.

## 9. 토너먼트 / 티어 (식물 개체별)

`TOURNAMENT_ROUNDS` / `RANK_TIERS`. 티어·점수는 **식물 한 마리마다** 따로 누적.

| 라운드 | 형식 | 통과 조건 | 보상 크레딧 | 승급 점수 |
|---|---|---|---:|---:|
| 예선 | 5판 다승 | 3승 | 60 | 0 |
| 16강 | 단판 | 1승 | 120 | 30 |
| 8강 | 단판 | 1승 | 200 | 50 |
| 4강 | 단판 | 1승 | 340 | 90 |
| 결승 | 단판 | 1승 | 600 | 160 |

**티어 승급 임계값** `RANK_TIERS`: 브론즈 0 → 실버 300 → 골드 700 → 플래티넘 1200 → 다이아 1900 → 마스터 2800 → 풀로세움 4000.

**상대 난이도** `buildEnemy`: `diff = 0.82 + 라운드×0.075 + 티어×0.05 ± 난수`. 라운드별 상대 생장단계 = 유체→성장체→성장체→성체→완숙체. 라운드 2부터 30% 확률로 포식형 등장.

〈밸런스 메모〉 적은 플레이어 최종 스탯에 diff를 곱해 생성하므로 항상 "근접 매치". 강화·티어가 오르면 상대도 같이 강해지는 상대 스케일링 구조.
