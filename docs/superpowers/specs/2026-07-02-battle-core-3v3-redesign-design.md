# 전투 코어 3v3 재설계 — 팀·심리전·손패 (🟡 브레인스토밍 중 · 사용자 승인 대기)

> **왜:** 현재 전투에 "의미 있는 결정"이 없다 — 1v1 고정이라 상성이 전투 중 결정이 아니고, 6스킬이 상시 열려 있어 아끼는 긴장이 없고, 봇은 읽을 패턴이 없다. 클래시로얄/포켓몬/양육 재료가 전부 껍데기로만 존재.

## 1. 턴 루프 (핵심 구조 변경)

현행: 내가 카드 누르면 즉시 해소 → 봇이 반응. **신규: 선택과 해소를 분리.**

```
[선택 페이즈]  양측 비밀 선택: 손패 4장 중 1장  또는  교체(대상 지정)
      ↓
[공개·해소]   ① 교체 선언 먼저 처리(포켓몬 룰 — 스왑 후 상대 공격은 새 식물이 받음)
              ② 스킬은 속도 순 해소(기존 선공 판정·applySkill·판정 연출 재사용)
      ↓
[턴 정리]     DoT/버프 틱(tickStatuses) → 에너지 재생 → 드로우(쓴 자리 채움)
              → 봇 큰 기술 예고 노출 → 다음 선택 페이즈
```

- 동시 공개는 기존 **judge overlay 3칸 슬롯**(내 카드=왼·화살표·상대=오른)과 자연스럽게 맞음 — 뒤집힌 카드 2장 → 동시 오픈 연출로 강화.
- 양측 다 교체하면 스킬 해소 없이 다음 턴.
- **결정론 유지:** 선택은 입력, 해소는 기존 `rng()` 시드 경로 그대로 → 서버 재검증(#8) 호환.

## 2. 팀 & 교체

- **출전 3마리** (보유 3 미만이면 있는 만큼, 1마리도 진행 가능 — 튜토리얼 자연 대응). 전투 시작 전 팀 편성(기본=최근 팀, `state.last_team`).
- **교체 = 그 턴의 행동.** 들어오는 식물이 상대 스킬을 받는다(구잇발 리스크). 교체는 항상 선택 가능(단 남은 식물 1마리면 불가). 기절 시 다음 투입은 **무료** — 해소 종료 직후 즉시 투입 선택(턴 미소모, 그 턴 상대 추가 행동 없음).
- 승리 = 상대 팀 전멸. 토너먼트 규칙(예선 5판3선 등)은 "판"(=한 경기)의 정의만 1v1→3v3로 바뀌고 판 수·보상·승급은 불변.
- 등장 훅: `onSwitchIn(unit)` 자리만 마련(아키타입/변이형 입장 효과는 후속 콘텐츠).

## 3. 덱 8 / 손패 4 / 에너지 커브

- **덱 = 식물당 로드아웃 8장**(`MAX_LOADOUT` 6→8). 변이 전용 스킬(무기 등 `variantSkillsOf`)도 **덱에 흡수**(하단 별도 바 폐지 — 손패 시스템과 이원화 방지, 횟수제 `uses`는 유지).
- **손패 4장**, 사용한 카드는 덱 맨 뒤로 → 즉시 다음 카드 드로우(CR 사이클). **"다음 카드" 미리보기** 표시.
- 핸드는 **식물별로 보존**(교체 후 돌아오면 이어서) — 핸드 관리가 교체의 전략 이유가 됨.
- **에너지 = 플레이어 공유**(교체해도 유지): **시작 2 · 매 턴 +1 · 최대 8** (1안, 튜닝 대상). 기존 스킬 코스트 0~3 그대로 호환 — "3코를 지금 쓸까, 모아서 다음 턴 콤보 칠까"가 생김.
- 손패가 전부 사용 불가(코스트 초과)면 폴백 액션 **"숨 고르기"(에너지 +1, 행동 안 함)** 상시 제공 — 데드락 방지.
- 시작 만땅(현행) 폐지. `energyGain` 스킬들은 커브 위에서 가속 수단으로 가치 상승.

## 4. 하이브리드 심리전 (봇을 "읽는" 재미)

- **아키타입 성향 tell (기존 14종 재활용):** `aiPickSkill`을 성향 가중치 기반으로 재작성 — 광폭=공격 편향·HP 무시 / 수호=HP 40%↓면 가드/회복 / 책략=디버프 선점 / 탐식=흡혈 우선 / 교활=교체 잦음 … 적 정보 패널에 **아키타입 배지 + 성향 한 줄**("성향: 저돌적") 노출 → 학습 가능한 패턴.
- **큰 기술 예고:** 봇이 다음 턴 코스트 3+ 스킬을 낼 예정이면 턴 정리 때 적 스프라이트에 **차오르는 이펙트 + ⚠️**(종류는 숨김) → 가드/교체로 대응하는 결정 창출.
- **PvP(후속):** tell 없이 순수 동시 선택 — 같은 루프에 정보만 빼면 됨(이중 시스템 없음).

## 5. 상성의 승격

- 배율 1.5/0.67 유지. 교체가 "수"가 되면서 상성표가 전투 중 실시간 결정으로 승격.
- 초보 배려: **팀 바에 후보별 vs 현재 적 상성 아이콘**(▲유리/▼불리) 표시. 약점 적중 연출 강화(기존 spriteFx 확장).

## 6. 전투 상태 구조 (`B`)

```js
B = {
  pTeam:[unit,unit,unit], eTeam:[...],   // makeCombatant 산출 — 유닛 구조 불변
  pActive:0, eActive:0,                   // 활성 인덱스
  pEnergy, eEnergy,                       // 공유 에너지(유닛 개별 energy 폐지)
  pHands:[[4장]…], eHands:[…], pDecks/eDecks, // 식물별 핸드/덱 사이클
  pIntent, eIntent,                       // 이번 턴 비밀 선택 {kind:'skill'|'switch', …}
  eForecast,                              // 봇 큰 기술 예고 상태
  turn, over, busy, mission?              // 기존 유지
}
```
- `makeCombatant`·`applySkill`·`tickStatuses`·상태이상·태그 시너지(#13)·속성 시그니처(#14) **전부 재사용**. 유닛 내부는 거의 무변경, 바깥 오케스트레이션(`playerSkill` 루프)을 "의도 수집 → 해소" 구조로 재작성.

## 7. 전투 화면 UI (이 스펙 범위 = 전투 화면만)

- 무대: 활성 1v1 표시 유지(현 레이아웃 재활용). 추가: **하단 팀 바 3슬롯**(HP·상성 아이콘, 탭=교체 선택), **손패 4장 + 다음 카드 + 에너지 바**(현 스킬바 영역 개편).
- 선택 페이즈: 카드 제출 시 뒤집힌 채 중앙 슬롯에 → 상대 카드와 **동시 오픈**.
- 홈/탐사/양육 등 전면 UI 개편은 3부 스펙.

## 8. 세이브/마이그레이션

- `normalizeState`: `equipped_skills` 6→8(부족분은 `plantKnownSkillIds`에서 자동 백필), `state.last_team` 신설, 유닛 `energy` 필드 → 공유 에너지로 이관(전투 중 세이브 없음이라 안전).
- 기존 식물/스킬/카드/등급 데이터 무손실.

## 9. 기존 콘텐츠 전환 (🟡 전면 전환안 — 승인 대기)

- **토너먼트:** 봇 팀 3마리 생성(`buildEnemy`×3 — 속성/타입 다양성 보장 규칙). 예선/본선/티어 구조 유지.
- **미션(#15):** `CAMPAIGNS` 적을 팀으로 재저작 — 초반 스테이지 1~2마리(빠른 판), 보스전 2~3마리(보스+호위). `enemy` → `enemies[]` 스키마 확장, 기믹은 보스 유닛에 부착(기존 그대로).
- **1v1 코드 제거**(병행 유지 없음 — 부채 방지). 항복/보상/승급 로직은 판 단위라 그대로.
- 대안(미채택): 토너먼트 먼저 전환 후 미션 후속 / 별도 모드 병행 — 정체성 분산으로 비추천.

## 10. 구현 단위 (writing-plans 입력용)

1. **팀/에너지/핸드 데이터 계층** — B 구조·마이그레이션·팀 편성 저장 (엔진 무접촉, `__test`)
2. **의도 수집·해소 오케스트레이션** — 선택 페이즈 분리, 교체 우선 룰, 속도순 해소 (핵심 공사)
3. **봇 AI 성향화 + 예고** — `aiPickSkill` 성향 가중치·교체 판단·`eForecast`
4. **전투 UI** — 팀 바·손패 4+다음 카드·에너지 바·동시 오픈 연출
5. **토너먼트/미션 3v3 전환** — 봇 팀 생성·CAMPAIGNS 재저작·1v1 제거
6. **검증** — 셀프테스트(핸드 사이클·교체 룰·에너지 커브·결정론) + preview E2E

## 11. 범위 밖 (후속 스펙)

- **2부 메타 루프:** 양육 선택→개체 전투 특화(비료 방향성), 팀 시너지 보너스, 수집 동기 재정렬.
- **3부 UI/연출 전면:** 홈="정원+팀" 중심 재구성, 도트 통일(#16 연결), 시작화면(#11).
- PvP 매치메이킹/서버(#8 순서 그대로), 등장 효과 콘텐츠, 관전 모드.
