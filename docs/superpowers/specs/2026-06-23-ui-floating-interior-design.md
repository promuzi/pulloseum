# 풀로세움 UI 리디자인 설계 — "함선 내부 장면 + 떠있는 도트 UI"

> 2026-06-23 확정. 브레인스토밍으로 합의한 UI 방향의 단일 설계서.
> 구현은 이 문서를 따른다. 상위 허브: [`../../master-roadmap.md`](../../master-roadmap.md) 결정 #11.

## 0. 한 줄 컨셉
각 화면 = **하나의 함선 내부(한 방)를 보는 장면**. 그 장면 위에 **HUD·버튼·선택지가 "떠 있고"**, 가운데 식물은 방 안 **보관 포드**에 놓여 관리·보관된다. 미니멀하되 **인디 도트 게임 특유의 톡 튀는** 포인트.

## 1. 핵심 원칙
- **헤더 ↔ 배경 경계 없음.** 딱딱한 흰 바(슬랩) 금지. 화면은 끊김 없는 한 장면.
- **배경 = 밝은 미래 타일 함선 내부.** 완만한 1점 투시로 **넓고 열린** 방(깊이 과하지 않게, 직사각 터널 금지). 가장자리 **은은한 음영**으로 너무 허옇지 않게.
- **UI = 장면 위에 부유.** 어두운 반투명 픽셀칩 + 청록/액센트 테두리 + **하드 도트 그림자**로 띄움.
- **중앙 식물 = 보관 포드/받침대** 위. 방 안에서 관리·보관되는 초점.
- **도트 교체 대비:** 배경은 나중에 `background-image`(도트 PNG)로 교체할 한 장 슬롯 구조. 식물/슬롯은 고정 프레임, 칩은 픽셀 보더. 기존 `SPRITE_OVERRIDES` 경로 유지.

## 2. 색 / 토큰
| 용도 | 값 |
|---|---|
| 선체 타일(천장/뒷벽/좌우벽/바닥) | `#eaf0f4` / `#dfe7ed` / `#d6dfe6`·`#cad4dc` / `#bcc8d0` (좌→우, 멀→가까이 음영) |
| 청록 액센트 / 딥 | `#2dd4bf` / `#0a8478` |
| 떠있는 칩 배경 / 테두리 | `rgba(10,18,28,.6)` / 2~3px 청록(또는 방 액센트) |
| 하드 그림자(부유감) | `box-shadow: 3px 3px 0 rgba(0,0,0,.4)` |
| 재화 골드 / 위험 빨강 | `#c9962f` / (체력 위험만 빨강 유지) |

## 3. 떠있는 UI 컴포넌트 규칙(공통)
- **HUD:** 좌상 레벨칩, 우상 재화칩 — 슬랩 없이 장면 위 부유.
- **하단 네비:** 바 없음. 떠있는 5개 픽셀칩, **가운데(식물·전투) 크게**.
- **액션 버튼(전투 등):** 청록 솔리드 + 진한 보더 + 하드섀도 → 톡 튐.
- **모든 칩:** 라운드 작게(4~6px), 픽셀 보더 2~3px, 하드 도트 그림자. 절제 — 포인트색만 강하게.

## 4. 방별 명세 (한 방씩 **수직 완성**)
1. **메인(출전 데크)** — 보관 포드의 식물 + 떠있는 HUD/전투/네비. 재배사 슬롯 = 떠있는 칩 줄.
2. **탐사(관측 갑판)** — 같은 방의 **뒷벽 관측창이 우주맵**으로 확장. 행성 = 떠있는 선택 노드.
3. **양육(식물원)** — 방 톤 초록 재배등, 화분 = 보관 포드 여러 개. 액센트 **초록**.
4. **상점(보급 단말기)** — 방 한쪽 벽이 큰 **스크린(스캔라인)**. 상품 = 떠있는 카드.
5. **함선(정비 베이)** — 방 내부를 도트로 도는 화면(기존 캔버스). 컨트롤 = 떠있는 칩.
6. **전투(스테이지)** — 몰입형 어둠 유지. HUD·카드만 떠있는 칩.
7. **관리/모달(강화·변이·스킬·가방)** — 떠있는 패널. 슬랩 탭 → 떠있는 탭.

## 5. "한 방 완성" 기준 (이거 다 되면 그 방 끝 — "헤더만 바뀜" 방지)
1. 배경 = 함선 내부 장면(플레이스홀더 OK) · 2. 헤더 슬랩 없음(HUD 부유) · 3. 버튼·네비 떠있는 칩 · 4. 식물 보관 포드 · 5. 액센트만 톡, 나머지 절제 · 6. 짤림 없이 정렬 · 7. (크롬 연결 후) 스크린샷으로 정렬·대비 확인.

## 6. 작업 순서
**메인 → 탐사 → 양육 → 상점 → 함선 → 전투 → 관리/모달.** 한 방을 100% 완성하고 다음으로.

## 7. 구현 메모 (기존 코드에서 교체할 것)
- 1·2단계에서 만든 **흰 슬랩 헤더를 부유 칩으로 전환:** `#topHeader`(상단 계기판 바) → 떠있는 좌/우 HUD 칩, `#bottomNav` → 떠있는 5칩, `.space-map-head`/`.nursery-top`/`#shipTopbar`/`#shopModal h2`(콘솔 헤더) → 장면 + 부유 요소.
- 배경 장면은 화면 컨테이너 뒤 레이어(SVG/CSS)로, 나중에 `background-image` 교체 가능하게 분리.
- 청록 액센트·도트 그림자·픽셀 보더는 공통 클래스로 묶어 방 간 통일.
