> ⚠️ 이 설계는 기존 로드맵 결정 일부를 **수정**한다. 핵심: "생장은 경험치로만, 물/비료 폐지"였으나 → **생장과 열매를 분리**하고 물/비료는 **열매 쪽으로 부활**(생장엔 여전히 무영향). (§7 결정 반영)

## 1. 핵심 모델 — 생장과 열매는 완전히 분리된 2축

| 축 | 동력 | 결과 | 비고 |
|---|---|---|---|
| **생장(진화)** | 경험치/레벨업 (전투·행동) | 6단계 진화(씨앗→…→완숙체) | 열매와 **무관**. 기존 시스템 |
| **열매(양육)** | **시간 경과** + **환경(물·비료·화분)** | 게이지가 차면 열매가 맺힘 → 수확 | 방치형. 미네랄로 즉시 가속 |

→ 경험치/레벨업은 **생장(진화)에만** 관여, 열매와 무관.

## 2. 양육 경제 — 교차 루프 (닫힌 자급 아님)

- **물/비료의 출처:** ① **탐사·전투 승리 보상**, ② **랜덤 소모품 상자**. (상점 크레딧 직접 구매 아님)
- 의도: 전투·탐사를 돌릴 동기가 양육으로 연결되는 **교차 루프**. 양육은 다른 시스템이 먹여준다.
- 무한증식 방지: 자원은 게임플레이로만 유입되므로 자연 상한.

## 3. 열매 게이지 모델 (핵심 메커니즘)

### 3-1. 맺힘 (단계 게이트 + 게이지)
- **성장체(growing)부터** 열매를 맺는다(성장체·성체·완숙체). 그 이전(씨앗·새싹·유체)은 **낙엽 = 하위 소모품**만.
- 식물마다 **최대 열매 개수 `maxFruits`** 가 정해져 있다(종·단계로 결정).
- **충전 게이지**가 시간에 따라 찬다. 게이지가 **꽉 차면 열매 1개가 맺힌다** → 게이지 리셋 → 다음 열매용으로 다시 충전.
- 맺힌 열매 수가 `maxFruits`에 도달하면 **충전 정지**, **게이지가 붉어진다**(꽉 참 신호).
- 각 열매는 **맺히는 순간 희귀도(색)가 롤·고정**되어, 식물에 **그 색으로 달린다**(수확 전에 무엇이 맺혔는지 확인 가능).

### 3-2. 단계 점증
- 단계가 높을수록: **등급 확률↑ · `maxFruits`↑ · 기본 주기↓**.

### 3-3. 환경 레버
- **물/비료 (활성 버프, 시간 감쇠):** 일정 비율 채울 때마다 게이지 **충전 속도↑** + 맺히는 열매 **등급 확률↑**. 시간이 지나면 효과가 **서서히 감쇠**(레벨이 줄어듦).
- **화분 (패시브 베이스):** **기본 충전 속도** + **`maxFruits` 보너스**. (전투 스탯 무영향 — 양육 전용)
- **즉시 가속(미네랄):** 유료 재화 미네랄로 게이지 즉시 충전/단축.

### 3-4. 수확
- **수확 버튼 = 현재 맺힌 열매 전부 수확.**
- 게이지가 중간까지 차 있으면 **초기화하지 않고**, 수확 후 그 진행도에서 **이어서** 충전한다.

## 4. 열매 = 5색 희귀도

- **희귀도 5단계(스킬과 동일 색):** 흰 < 초록 < 파랑 < 보라 < 주황.
- 색의 표현 = **테두리 + 은은한 글로우**. **열매 이미지(모양)는 통일** — 희귀도 **색만** 차이. (※ "타입/속성별 모양" 안은 폐기/향후 확장으로 보류)
- **색 = 등급, 내용은 풀에서 롤:**
  - **흰/초록 = 소모품**
  - **파랑/보라 = 변이 카드**
  - **주황 = 본인 전용 스킬 또는 고등급 변이 카드**

### 4-1. 보상 규칙 (열매 1개당)
- 열매 1개 = **보상 1종류**.
- **변이 카드·스킬은 1개**.
- **소모품은 한 종류라도 여러 개** 나올 수 있다(예: 물 ×3). (기다려 깠는데 물 1개는 허무하므로 묶음 산출)
- 보상 성격:
  - **소모품** = 주 산출물(양육 재료·생장 촉진제 등).
  - **변이 카드** = 그 식물 변이형 관련, **다른 식물에도 장착 가능**(전파/수집 공급원).
  - **본인 전용 스킬** = 그 식물 **전용** 스킬 풀 확장(자기강화, 전파 X).

### 4-2. 본인 스킬 풀
- **전투 로드아웃은 6칸 고정.**
- 보유 스킬 풀은 **무제한 수집**. **중복 획득 시 크레딧으로 전환**(가챠식 중복 보호).
- (향후 스킬 종류를 대폭 늘려 다양성·수집 재미 확대 예정.)

## 5. ⭐ 공통 RewardReveal 연출 (신규 횡단 컴포넌트)

상자/열매를 **터치해서 여는** 공통 개봉 연출로 통일한다(클래시로얄식). 셋 다 같은 패턴:

| 사용처 | 컨테이너 모양 | 비고 |
|---|---|---|
| 상점 랜덤박스 | **보급 박스** | 기존 보급상자 |
| 탐사 결과 | **탐사 상자** | 기존 탐사 보상 창 대체 |
| 열매 수확 | **열매** | 희귀도 색 적용 |

**패턴:** 특정 배경 위에 컨테이너가 등장 → **터치하면 열림** → 보상 공개. 여러 개면 **순차 개봉**(하나 터치→열림→다음 터치→반복).

→ 재사용 컴포넌트 `openRewardReveal({ kind, items })` 로 추출. 기존 보급상자/탐사결과 창도 이 컴포넌트로 리팩터.

## 6. UI / 데이터 / 구현 메모

### 6-1. 양육 페이지 (하단 4번째 탭)
- 화분 + 식물(누끼) + **식물에 달린 열매들(색별)** + **충전 게이지**(남은 시간) + **물/비료 투입** + **수확 버튼** + **미네랄 즉시 가속**.
- 게이지 색: 물/비료 레벨에 따라 **그라데이션으로 진해짐**, 꽉 차면 **붉음**.
- 종자 가방 → 양육 입구(심기 → 화분 배치 → 양육 시작).

### 6-2. 데이터 모델
- 식물 개체에 `fruit` 상태 필드 신설:
  - `gauge` (0~1 현재 충전), `maxFruits`, `ripe[]`(맺힌 열매 = 각 `{rarity}`), `lastTick`(시간 계산 기준),
  - `careWater`/`careFert`(활성 버프 레벨, 감쇠), `potQuality`(화분 베이스).
- `normalizeState` 마이그레이션으로 기존 세이브에 기본값 주입(무회귀).
- 시간 경과 계산은 **틱 기반**(마지막 접속 시각 대비 경과로 게이지·감쇠 일괄 적용 — 방치형).

### 6-3. 식물/화분 2레이어 (#12·#3 전제)
- 현재 `spriteFor`는 식물만 그리고 화분은 배경. 식물 PNG + 화분 PNG **2레이어**(화분 입구 앵커, idle sway는 식물 레이어만)로 분리.

### 6-4. 연출 재활용
- 수확/개봉 연출은 §5 공통 컴포넌트 + 기존 `#evolveModal` 패턴 참고.

## 8. 기존 구현과의 관계 (⚠️ 신규 구축 아님 — 개편)

이미 양육(식물원) 시스템이 **부분 구현**돼 있다. 본 설계는 이를 **개편**한다.

**현재(개편 전):**
- 물/비료 = **일일 카운터**(물 20회/비료 10회, 자정 리셋 — `state.nursery_daily`).
- 열매 = **누적 게이트**(물 10 + 비료 5 누적 → 열매 1~3개), `p.nursery.{water,fert,stage,fruitCount,fruitReady}`.
- 열매 보상 = **스킬 박스 only**(`nurserySkillReward` — 생장단계 스킬 풀에서 1개). 수확 → `state.nursery_fruits` → 가방에서 개봉.

**개편 후(본 설계):**
- 물/비료 = **인벤토리 자원**(탐사·전투·상자 보상) + **감쇠형 활성 버프**(일일 카운터 폐기).
- 열매 = **시간 게이지 + `maxFruits`** (누적 게이트 폐기). 맺힐 때 **희귀도 색 롤**.
- 보상 = **5색 희귀도 → 소모품/변이카드/본인스킬**(스킬 only 폐기). **수확 후 게이지 보존.**
- 개봉 = **공통 RewardReveal**(가방 개별 개봉 → 공통 연출로 통일).

**현재 코드 위치 (개편 대상):**

- 양육 탭: `openNursery` [9956], `renderNurseryGrid` [9970], `openNurseryDetail` [10007]
- 열매 로직: `checkNurseryFruitReady` [10082], `doHarvestPot` [10094]
- 수확 가방/개봉: `openFruitBag` [10114], `clearOpenedFruits` [10146], `nurserySkillReward` [10175]
- 스프라이트/화분: `spriteFor()` [7706], `SPRITE_OVERRIDES` [7590], `.pot-slot`/CSS [1310~1384]
