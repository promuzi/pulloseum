## 1. 시스템 규칙 (전역 — 전 변이형 공통)

1. **변이형 = 패시브 없는 "카드 슬롯 게이트".** 변이형 자체는 전투에 직접 효과 없음. 그 변이형을 가지면 **해당 계열 변이 카드를 장착할 수 있는 폭**이 생기는 것만이 의미. 대신 **카드 하나하나의 특색을 뚜렷하게.**
2. **식물당 변이형 1개**(또는 무변이). **개체당 변이 2개 이상(다중 variantSlots)은 폐기** — 스킬 늘어나는 변이가 겹치면 너무 복잡. (카탈로그 `variantSlots` 다중 배분 아이디어 무효.)
3. **무변이 = 공통 카드만** 장착 가능. **변이형 보유 = 공통 + 자기 계열 카드** 장착. 그게 "폭"의 전부.
4. **변이 카드 슬롯 최대 5칸**(아이템 `unlock_trait_slots`로 순차 해금, 기존 유지).
5. **같은 `card_id`는 등급 무관 1장만** — 새로 장착 시 기존 같은 id 카드를 **자동 교체**(앞서 등급 다른 중복 장착 가능했던 버그 수정).
6. **무등급(`fixed`) 카드 클래스 신설** — 등급 롤(S~D) 없이 **고정 효과**(강한 개성·도박·기믹). 데이터상 `fixed:true`로 등급 스케일/서브특성 스킵, 단일 효과.
7. **포식형 무료 기본기 제거** — 포식의 "공짜 위력150 기본스킬"을 없애고, 포식 스킬은 약한 기본기로 낮춘 뒤 **카드로 강화**(아래 ②). → 전 변이형 대칭, 밸런스 이슈 #7 해소.
8. 효과 **하단 "변이 스킬" 바**: 일부 변이형(포식·무기·독성)은 상단 6칸 로드아웃과 **별개로 하단 가로 바**에 변이 스킬을 둠. **구조는 변이형마다 다름**(공통 구조 아님).

## 3. 상태이상 메커니즘 (독성·포자 설계의 전제)

코드 확인 결과(`index.html` line ~9871·9888):
- DoT(중독/출혈/화상)는 적용 시 `dots` 배열에 **별도 스택으로 push** → 매 턴 종료 시 **모든 스택 합산 피해**, 각 스택은 자기 지속턴 후 개별 소멸.
- 즉 **이미 스택형** — 걸 때마다 +1 스택 = 턴당 피해↑. "한 번 걸면 끝"이 아님. **단, 현재 스택 상한 없음** = "독이 너무 강함"의 근본 원인.
- DoT는 **방어 무시**(고정 피해). **마비는 DoT 아님** — 다음 턴 1회 스킵 단발 제어(`skipNext`).

**결정 — 스택 상한제 도입:** 독 최대 **N스택**(예 4), 턴당 피해 = 스택 × (atk×%). → "여러 번 묻히는 의미(상한까지 빌드)"와 "독 통제(상한)"를 동시에 해결.

**결정 — 상태이상 적용 = 하이브리드(대부분 확정, 강제어만 확률):**
| 효과 | 적용 | 이유 |
|---|---|---|
| DoT 데미지(중독·출혈·화상) | 적중 시 **100% 확정**(스택+1) | 누적 빌드 — 확률 끼면 운빨 |
| 스탯 디버프(공/방/기동↓) | 적중 시 **100% 확정** | 온건 — RNG 최소화 |
| 강제어(마비·기절) | **자체 확률**(예 25~30%) | 100%면 스킵 도배로 게임 종료 |
> 한 스킬에 여러 효과(강산=독+방깎) → 적중 시 독·방깎 동시 확정, 마비 섞이면 그것만 따로 확률.

**속성 보완:** **브레스 = 식물 속성 축 발현**(자기 속성으로 공격, 상성 + 속성별 rider). 비늘은 타입 종속 없는 단순 방어(사용자 결정: 타입과 안 묶음).

**하단 바 2스킬(보유 시 기본):**
| 스킬 | 효과 | cats |
|---|---|---|
| 🛡️ 비늘 강화 | N턴 방어 태세 — 받는 피해 −X% + 매 턴 **비늘 스택** 누적(점점 단단, 타입 무관) | 🛡️✨ |
| 🔥 브레스 | **자기 속성으로 발현**하는 공격(상성 적용) — 속성별 rider(불=화상/빙결=둔화/번개=치명/풀=흡혈/대지=방무시 …, 기존 ELEMENT rider 재사용) | 🗡️ |
> 브레스는 식물이 *이미 가진* `element`로 발현 → 외부 속성 부여가 아니라 자기 속성 표현이라 §5 "무기 속성 부여(미래)"와 충돌 없음.

**브레스 강화 카드(등급제) 4:** 🔥융합 브레스(위력+X%) · ⚗️원소 농축(속성 rider 효과 강화) · 🌫️광역 브레스(광역화/명중↑) · 💥관통 브레스(방어 일부 무시)
**비늘 강화 카드(등급제) 4:** 🛡️강철 비늘(방어 효과+X%) · ♻️재생 비늘(유지 중 매턴 체력회복) · 🔺가시 비늘(피격 시 반격) · 🏔️원소 비늘(자기 속성 약점 피해 경감)
**투트랙 시너지 카드 2:** 🐉용의 분노(비늘 스택 높을수록 브레스 위력↑) · 🔄비늘 연소(브레스 시 비늘 스택 소모→추가 위력)
**무등급 2:** 🐲고룡의 위엄(전투 시작 비늘 3스택 + 첫 브레스 위력 2배) · 🔥멸망의 숨결(브레스가 적 체력 비례 추가피해, 쿨다운 김)

## 5. 후속 정리 (박제 — 나중에 한 번에)

1. **4축 분리 원칙 못박기:** 상성=속성 / 상태이상=독·포자 / 물리=무기 / 스탯=공통. 각 시스템은 자기 축만 소유. → "무기 속성 부여"(미래)는 **상태이상이 아니라 상성(`element`)을 주는 것**으로 설계하면 독성과 필드가 달라 안 겹침.
2. **무기 카드에서 상태이상 제거:** 🔥원소칼날(화상)·🗡️출혈날 등은 무기(물리 축)가 상태이상 축을 침범 → 화상/출혈은 독성으로 몰고 무기는 물리만. (기절만 망치의 물리 충격으로 예외 허용.) "왕창 넣고 트림" 기조로 **일단 두었다가 한 번에 정리**.
3. **개체-변이 분류(미래):** 변이에 따라 다른 개체로 분류 예정. 시그니처(개체 고유) vs 카드(범용)를 **범위로 구분** — 같은 효과 중복 방지.
4. **`__catalogSelfTest` 규칙 추가:** 한 카드가 같은 축을 두 번 걸지 않는지 검증.
