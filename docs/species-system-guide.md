# 풀로세움 — 식물 종(속성 × 타입) 공략집

> **핵심:** 모든 식물의 기본 스탯 = **속성 수치 + 타입 수치**.
> 속성(7종)은 상성·색·전투 성향을, 타입(5종)은 몸체(실루엣)·체격 성향을 담당한다.
> **5타입 × 7속성 = 총 35종.** 생장단계별 외형은 절차적으로 자동 생성된다.
>
> 기준일: 2026-06-18 · 출처: `index.html` (`ELEMENT_STATS` / `TYPE_STATS` / `SPECIES_GRID` / `SPECIES`)

---

## 3. 속성 상성 (개정판)

`eff(a,d)` — 공격 속성 `a`가 방어 속성 `d`를 때릴 때의 배율. 강점 **1.5×**, 약점 **0.67×**, 그 외 1×.
(코드상 `eff`는 `strong` 배열만 사용하며, 한쪽이 강하면 반대쪽은 자동으로 약점이 된다.)

| 공격 속성 | 강하게 때림(1.5×) | 약점(1.5× 받음) |
|---|---|---|
| 🔥불 | 풀 · 빙결 | 물 · 바람 |
| 💧물 | 불 · 대지 | 풀 · 번개 · 빙결 |
| 🌿풀 | 물 · 대지 | 불 · 바람 · 빙결 |
| 🪨대지 | 번개 · 바람 | 풀 · 물 |
| 🌪️바람 | 풀 · 불 | 대지 |
| ⚡번개 | 물 | 대지 |
| ❄️빙결 | 물 · 풀 | 불 |

### 이번 개정 내역
- **대지** 강점에서 **빙결·불 제거** → 강점이 번개·바람 2종으로 축소(과한 커버리지 완화).
- **풀** 약점에서 **번개 제거**(번개가 더 이상 풀을 1.5×로 못 때림).
- **바람** 강점에 **불 추가** → 바람이 불을 1.5×. (바람 보정의 핵심)
- **불** 강점에서 **바람 제거**(불 vs 바람은 이제 0.67×, 바람이 우위).
- **빙결** 강점에서 **번개 제거 + 풀 추가** → 빙결이 풀을 1.5×, 번개엔 평타.

〈밸런스 메모〉 **번개**는 강점이 물 1종뿐이라 이제 공격 커버리지가 가장 좁다(공격 17로 높은 화력으로 상쇄). 반대로 **물·풀**은 1.5×로 맞는 약점이 3종이라 운용 시 상성 주의.

## 4. 이미지(외형) — 생장단계별 자동 생성

식물 외형은 그림 파일이 아니라 **절차적 SVG**로 만들어진다(`composePlantBody` / `composePlantSvg`).
- **입력:** 타입(실루엣) × 속성(색·모티프) × 생장단계(크기).
- **생장 6단계:** 씨앗 → 새싹 → 유체 → 성장체 → 성체 → 완숙체. 단계가 오를수록 실루엣이 자라고, 완숙체엔 오라+✦가 붙는다.
- 따라서 **35종 × 6단계 = 210개 외형이 전부 자동 지원**된다. 별도 PNG 추가 불필요.
- 특정 조합만 손그림으로 교체하려면 `SPRITE_OVERRIDES[`타입_단계_속성`]`에 이미지 URL을 넣으면 된다.

타입별 실루엣 요지:
| 타입 | 실루엣 |
|---|---|
| 🌿 초본형 | 여러 갈래 잎날이 부채처럼 펼쳐짐 |
| 🌳 목본형 | 줄기 위 둥근 수관(나무) |
| 🌸 화초형 | 봉오리→개화(꽃잎 방사) |
| 🌵 다육형 | 통통한 기둥 + 가시·곁가지 |
| 🍃 덩굴형 | 휘감기는 덩굴 줄기 + 작은 잎 |

## 5. 개발자용 — 수치를 만질 때 (index.html)

| 만지고 싶은 것 | 상수/함수 |
|---|---|
| 속성별 스탯 기여 | `ELEMENT_STATS` |
| 타입별 스탯 기여 | `TYPE_STATS` |
| 종 목록(이름·타입·속성) | `SPECIES_GRID` (→ `SPECIES` 자동 생성) |
| 속성 상성 배율 | `ELEMENTS[].strong` / `eff()` |
| 종→외형 타입 매핑 | `SPECIES_SEEDTYPE`(SPECIES에서 자동 파생) |
| 외형 생성 | `composePlantBody()` / `composePlantSvg()` |
| 손그림 교체 | `SPRITE_OVERRIDES` |

> 종을 추가/수정하려면 **`SPECIES_GRID`에 `[key, 이름, 타입, 속성]` 한 줄**만 넣으면 스탯·외형·적 등장·상성이 자동 연결된다. (기존 7종 key: flame/aqua/thorn/gaia/windy/spark/frost — 세이브 호환 위해 유지)
