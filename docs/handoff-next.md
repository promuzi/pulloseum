## 4. 다른 기기에서 재개하는 절차
1. 로컬 풀로세움 폴더에서 최신 받기:
   ```
   git pull origin main
   ```
   (백업 브랜치도 있음: `claude/colosseum-game-update-vsgw8z`)
2. 새 기기/클론이면 1회: `git config core.hooksPath .githooks` (문서 동반 푸시 강제 훅).
3. AI(Claude/Codex 등)로 폴더 열면 → **이 파일 → [master-roadmap.md](master-roadmap.md) → [feature-designs.md](feature-designs.md)** 순으로 읽으면 맥락 복원 끝.
4. 게임 확인: `index.html` 더블클릭. (세이브는 브라우저 localStorage라 기기마다 따로인 점 주의.)
