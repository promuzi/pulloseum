|---|---|---|
| **장르 정체성** | 방치형이라 표방하지만 실제론 **수동 전투 중심** | AFK Arena/Journey: *진짜* 방치(오프라인 자동 성장·보상) |
| **수집** | 35종 + 변이카드, 상단 컬렉션 바 | Monster Super League: 도감·진화 3단계·속성 상성 + **도감 완성 보상** |
| **런/엔드게임** | 토너먼트 단판, 모드 탭 🔒 | PokéRogue: **끝없는 런 + 메타 진행**("죽고 더 강해지기") |

| **번식/육성** | 양육=물/비료로 열매(스킬 1개) | Ooblets/몬헌스토리즈: **교배·알 부화로 형질 유전** |

## 1. 제안 — 골라주세요 (각 항목 1개 선택)

### ① 방치(오프라인) 진행 — *최우선 추천* ⭐
> 간판이 "방치형"인데 닫으면 아무 일도 안 일어난다. 경쟁작 리텐션의 핵심.

- [ ] **A. 양육 오프라인 자동 진행** — 화분 식물이 닫혀 있는 동안 시간에 비례해 물/비료(또는 열매 게이지)가 차오름. 접속 시 "그동안 ○○ 모임" 수령 팝업. (가장 가볍게 시작)
- [ ] **B. 자원 자동 수급** — 시간당 크레딧/미네랄/종자 일정량 누적(상한 있음). 함선 가구로 효율↑(로드맵 #4 연동).
- [ ] **C. A+B 둘 다** (풀 방치 루프)
- [ ] **D. 안 함** (수동 유지)

### ② 매일 돌아올 이유 (리텐션 루프)
- [ ] **A. 일일 퀘스트 3종** (전투 N승 / 물주기 N회 / 탐사 1회 → 크레딧·보급상자 보상)
- [ ] **B. 출석 보상** (연속 접속 7일 캘린더, 7일째 변이카드)
- [ ] **C. A+B 둘 다** *(추천)*
- [ ] **D. 안 함**

### ③ 엔드게임 — 잠긴 "모드 🔒" 탭 채우기
> PokéRogue식 "끝없는 런"이 식물·생장 시스템과 궁합이 좋음.

- [ ] **A. 무한 콜로세움(로그라이크 런)** — 연승 도전, 라운드마다 강해지는 봇 + 라운드 사이 **임시 강화 선택지**(스킬/스탯 카드 3중 택1). 패배 시 도달 라운드만큼 메타 재화 획득 → 영구 강화. *(추천)*
- [ ] **B. 일일 시련** — 매일 바뀌는 특수 규칙(예: 화상 2배) 단판, 등수 보상.
- [ ] **C. 둘 다**
- [ ] **D. 다른 것(예: 보스 레이드)** — 메모로 알려주세요.

### ④ 식물 번식/교배 — 테마 강화
> "식물 키우는 게임"인데 정작 번식이 없음. Ooblets/몬헌스토리즈의 강점.

- [ ] **A. 2종 교배 → 새 종자** — 부모의 속성/변이형/잠재력 일부를 확률 유전. 양육 탭에 "교배" 추가. *(추천, 양육 루프와 직결)*
- [ ] **B. 열매 → 종자 회수** — 현재 열매(스킬 지급)에 더해 "씨앗 맺기" 선택지(같은 종 + 변이 가능).
- [ ] **C. 둘 다**
- [ ] **D. 안 함**

### ⑤ 전투 편성 — 1마리 → 파티?
> 가장 큰 구조 변경. 신중히 선택.

- [ ] **A. 현행 유지(1마리)** — 클래시로얄식 단일 영웅 정체성 유지. *(안전)*
- [ ] **B. 2~3마리 파티 + 교체** — 전투 중 식물 스위치(속성 상성 운영). 큰 작업이지만 깊이↑.
- [ ] **C. 부가 효과만** — 후보 식물이 패시브 버프만 제공(전투엔 1마리, 벤치는 응원). 절충안.

### ⑥ 수집 도감 & 완성 보상
- [ ] **A. 종 도감** — 35종 수집 현황 + 종별 도달 시 소량 영구 보상(미네랄 등), 100% 시 칭호/프로필.
- [ ] **B. 변이카드 도감** — 카드 등급별 수집률.
- [ ] **C. 둘 다** *(추천, 기존 컬렉션 바 확장이라 저비용)*
- [ ] **D. 안 함**

### ⑦ PvP (로드맵 #8)
- [ ] **A. 비동기 고스트 PvP** — 상대 로드아웃 스냅샷을 봇이 재생, 랭킹전. 서버 부담 적음. *(추천)*
- [ ] **B. 실시간** — 백엔드 필요, 후순위.
- [ ] **C. 지금은 보류**

### ⑧ 편의성(QoL) — 여러 개 선택 가능
- [ ] 자동 전투 / 전투 2~3배속
- [ ] 양육 "전체 물주기" 한 번에
- [ ] 보급상자 10연속 개봉 + 결과 요약
- [ ] 식물 목록 정렬/필터(속성·등급·생장단계)
- [ ] 전투 후 결과 스킵/빠른 진행

## 2. 바꾸면 좋을 점 (방향 제안 — 동의 여부만)

- [ ] **경제 곡선 점검** — 레벨업 물약이 현재 *무료·무한*(테스트용). 출시 전 정식 비용/획득처로 전환. (밸런스 [balance-sheet.md](balance-sheet.md) §8 연동)
- [ ] **튜토리얼/온보딩** — 첫 실행 시 탐사→양육→전투 한 바퀴 가이드. 신규 이탈 방지.
- [ ] **잠재특성 패시브 활성화** — 현재 20종이 "스킬 해금 분류"로만 쓰이고 패시브는 미반영(예약). 활성화로 빌드 다양성↑. ([trait-growth-roadmap.md](trait-growth-roadmap.md))
- [ ] **사운드 도입**(로드맵 #6) — 무음 게임은 체류시간 손해. BGM 1 + 효과음 몇 개만으로도 체감 큼.
